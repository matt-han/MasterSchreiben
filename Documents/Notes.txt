- Speichern der Parameter in *.testona Datei nicht am Anfang machen. Java-Object Tag wird serialisiert 


JAVA - DOORS
============
Aufruf von BM_PARAMETERS_MODULE_NAME über Java ergibt den Pfad zu den Parametertabelle in DOORS.
So können Parameter aus DOORS geladen werden.
